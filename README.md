# Invoke-BuildAnonymousSMBServer
Use to build an anonymous SMB file server.

This is useful for testing CVE-2021-1675 and CVE-2021-34527.

Test is successful on the following system:

- Windows 7
- Windows 8
- Windows 10
- Windows Server 2012
- Windows Server 2012 R2
- Windows Server 2016
